# 开始训练！AI会看这20张图，自己学会分辨

## 🎉 第四步：测试你的AI！让它认一张新照片

训练完后，再新建一个代码框，粘贴：

```python
# 测试AI认不认得新图
from PIL import Image
import requests
from ultralytics import YOLO

# 加载训练好的模型中最好的一个
model = YOLO("runs/classify/train/weights/best.pt")

# 下载一张新的猫或狗照片（你也可以上传自己的）
response = requests.get("https://tinyurl.com/ai-test-cat")
with open("test.jpg", "wb") as f:
    f.write(response.content)

# 让AI看这张图
img = Image.open("test.jpg")
results = model(img)

# 输出结果
print("AI认为这是：", results[0].probs.top1)
img.show()  # 弹出图片给你看
```

点击 ▶️ 运行！

你会看到：

- 一张新图片弹出来（可能是猫，也可能是狗）  
- 下面写着：**“cats” 或 “dogs”**
